# Smart-Mess

Auth API

POST /auth/register

POST /auth/login

POST /auth/logout

User API

GET /users/{id}

PUT /users/{id}

DELETE /users/{id}

Meal API

POST /meals (add meal)

GET /meals (list meals)

PUT /meals/{id} (update meal)

DELETE /meals/{id}

Attendance API (who ate today)

POST /attendance

GET /attendance?date=YYYY-MM-DD

Expense API

POST /expenses

GET /expenses

PUT /expenses/{id}

DELETE /expenses/{id}

Bill/Payment API

POST /payments

GET /payments

GET /payments/{userId}
